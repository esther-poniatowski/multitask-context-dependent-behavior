# MTCDB - Architecture

## Top-Level Directory Structure

```tree
mtcdb/
├── config/              # Configuration files for the ETL process
├── data/                # Directory for storing raw and processed data
├── docs/                # Documentation files
├── src/                 # Source code for the ETL process
├── tests/               # Unit tests for the ETL process
├── scripts/             # Scripts for setup tasks
├── pyproject.toml       # Project metadata and dependencies
├── env_spec.yml         # Specification file for external dependencies (GitHub repositories)
├── environment.yml      # Conda environment file for runtime dependencies
├── environment-dev.yml  # Conda environment file for development dependencies
├── mtcdb.code-workspace # VSCode workspace file
├── LICENSE              # License file for the project
├── README.md            # Project overview and installation instructions
└── ROADMAP.md           # Project roadmap and future plans
```
